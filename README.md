📌 Project Overview

One Rep Max is a modern fitness e-commerce website developed using React.js. The website is designed for selling premium supplements and fitness products. The project focuses on creating an attractive and professional user interface without backend functionality.

---

🚀 Features

✅ Modern Dark Theme Design

✅ Responsive Homepage Layout

✅ Navigation Bar

✅ Hero Section

✅ Product Categories Section

✅ Product Cards

✅ Testimonials Section

✅ Footer Section

✅ Multiple Pages using React Router

✅ Component-Based Architecture
